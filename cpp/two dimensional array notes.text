two dimensional array:-The two dimensional array has two subscripts(indexes),one for rows and another one for columns.two dimensional array  is also called matrix.

Declaration of 2d array:-
datatype arrayname[rows][cols];

E.g.,
    int a[3][4]; // this array can store 3*3 matrix (3 rows and 4 columns)

Initialization of matrix : -E.g.,

    int a[3][3] = {
        1, 2, 3,

        4, 5, 6,
        7, 8, 9}
